CentOS 安装与配置

2020-07-16 linux

CentOS (Community Enterprise Operating System) 也即社区版的企业操作系统，是 Linux 发行版本之一，来自于 Red Hat Enterprise Linux 所开放的源码编译而成。因此，对稳定性较高的服务器通常以 CentOS 替代商业版的 Red Hat 。

两者的不同在于 CentOS 并不包含封闭源代码软件。下面会介绍桌面版的 CentOS 在安装时需要作的一些常用配置。

安装 CentOS #

对于 CentOS 桌面版的安装也比较简单，完全是基于图形界面的，安装时可以从 www.centos.org 上下载相应版本，一般用 LiveGNOME 或者 LiveKDE 。

当然，也可以从国内的一些镜像网站上下载，例如 阿里云镜像、搜狐镜像、网易镜像 上下载。

下载 ISO 镜像后可以通过如下方式安装到 U 盘，如果有 Linux 发行版，安装过程将很简单；如下是通过 dd 命令直接复制即可。

----- 查看U盘的设备号
# fdisk -l
... ...
Disk /dev/sdc: 4002 MB, 4002910208 bytes, 7818184 sectors
... ...

----- 将ISO镜像原样复制到U盘中
# dd if=xxx.iso of=/dev/sdb

----- 从另外一个终端执行，查看复制到U盘的进度
# while killall -USR1 dd; do sleep 5; done

在重启之后进入到 BIOS 启动界面，然后选择从 USB 启动进入安装界面。

安装 xWindows #

有些环境例如虚拟机可能是从 Mini 镜像安装的，此时一般没有安装图形界面，可以通过 yum groupinstall "GNOME Desktop" 命令安装即可，然后设置启动模式。

----- 查看当前模式，一般是multi-user.target，即命令行模式
# systemctl get-default

----- 修改为图形界面模式
# systemctl set-default graphical.target

然后重启即可。
