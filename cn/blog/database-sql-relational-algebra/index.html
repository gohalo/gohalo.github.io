符号名称含义备注
$\sigma$Select选择从关系中筛选部分行，形成新的关系
$\Pi$Project投影从关系中筛选部分列，形成新的关系
$\cup$Union并合并两个关系
$\cap$Intersection交取两个关系的交集
$-$Difference差取两个关系的差集
$\times$Product叉积两个关系的笛卡尔积
$\Join$Join联结两个关系在满足某些条件下的连接
$\rho$Rename重命名重命名关系中的列
$\leftarrow$Assignments赋值将某个临时查询命名为新关系
$\delta$Duplicate Eliminating去重
$\gamma/\Gamma$Aggregation聚合对部分列聚合形成新的关系
$\tau$Sorting排序排序结果形成新的关系
