Ray 源码解析

2024-08-16 ai linux

简介 #

以如下的示例为例。

import ray

ray.init(num_cpus=4)

@ray.remote
def hello():
    return "Hello World!!!"
oid = hello.remote()
print(ray.get(oid))

上述的 @ray.remote 实际上是一个装饰器，详细可以参考 Python 闭包和装饰器 中的介绍。
