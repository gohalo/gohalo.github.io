BootAgent

当管理的机器达到上万甚至是百万的数量级时，即使 Agent 的部署成功率能达到 99% 那也会导致有几百台的机器离线，如果 Agent 频繁变更可能会导致离线机器更多。

BootAgent 就是为了管理各个 Agent ，同时为了保证机制简单，可以使功能快速稳定；该 Agent 会在装机时安装，用来负责安装、升级、监控、重启其它 Agent 。
