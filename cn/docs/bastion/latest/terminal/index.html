简介 #

详细可以查看 SSHShellExecHandler() 中的实现，其中 Shell 相关的操作与命令行类似，通过 urfave 实现，可用命令详见 getNewApp() 函数的实现。

为了支持一些定制化的特性，可以对用户、用户组 (优先级高) 进行一些简单的配置，详细可以查看 配置相关-用户配置 中 shell 相关的介绍，当前用户变量会保存在 APP.Metadata 中。

查看状态 #

可以查看当前的一些系统、鉴权信息。

----- 查看当前系统信息
config> show system
    Version: v0.0.1 01ddfd47(integration)
System info: linux/amd64 go1.21.5
   Hostname: debian
       CPUs: 8
Go routines: 18
System Time: 2024-11-13T20:34:38+08:00
     Uptime: 4.830902472s

----- 当前用户的鉴权信息，其中 Policy 显示为：名称 Action Resource
config> show auth
Auth: auto(cn-foobar)
Roles:
    supervisor
    admin
Policies:
    DefaultAdminPolicy [Bastion:*:*] [Bastion:*:*:*:*]

同时也支持变量的设置。

----- 查看所有的会话变量
config> show variables
debug=false                 // 打印调试信息
format=thin-table           // 设置显示格式，目前只支持thin

----- 查看某个特定的变量
config> show variable format
format=thin-table
config> show variable test  
error: unsupport variable 'test'.

----- 设置变量
config> set format=thin-table

加解密 #

允许对某些配置参数等进行加解密操作。

----- 加/解密字段，允许有多个参数 [admin, super]
config> tool encrypt <YourPlainText> <MaybeSomePassword>
config> tool decrypt <YourSecretText>

----- 回显字符串，通常用于健康测试使用
config> echo "hello world"
----- 退出，也可以 Ctrl-D
config> exit
----- 查看帮助文档
config> help
