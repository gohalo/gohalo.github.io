登录失败 #

登录类型 #

通过 SSH 登录分成了几类，可以从日志的登录用户名和登录方式进行判断：

  • Shell 登录内置的 Shell 实现。
  • Invite 通过邀请码绑定用户对应的公钥信息，绑定信息后立即退出。
  • Bastion 可以将主机名作为登录用户使用，或者在 Shell 中选择对应的主机。

用户证书 #

通过 ssh 登录时，如果没有显示指定私钥，那么可能会选择系统下的某个密钥对，可以通过如下方式查询。

----- 查询某个用户关联的 fingerprint 信息
SELECT
    user.uid, user.name, login.id AS login_id,
    login.fingerprint, login.authorized
FROM user
JOIN user_login_key AS login
ON user.id = login.user_id
WHERE user.uid = "admin";

其中包含了证书的签名 (fingerprint) 以及公钥信息 (authorized)，最简单的，直接在 ~/.ssh 目录下 grep 查询到的公钥信息基本就可以确定实际使用的密钥对。

当然，也可以通过 ssh-keygen -E sha256 -lf id_rsa.pub 查看怀疑文件的签名信息，只是略微复杂。

后续会支持密钥对的重新绑定，不过当前可以手动修改。

DELETE FROM user_login_key WHERE id = 1;
UPDATE user SET invite='RmfNjMuOk5OnKGbH', status=0 WHERE id = 1;
UPDATE user SET invite='RmfNjMuOk5OnKGbH', status=0 WHERE uid = 'm00228901';

权限管理 #

用户会绑定到某个用户组，而用户组会绑定对应的角色或者策略。

如下用于用户、用户组的关联信息查询。

----- 某个用户所属的用户组
SELECT u.uid, u.name, g.name
FROM user AS u
JOIN map_user_group ON u.id = map_user_group.user_id
JOIN user_group AS g ON map_user_group.user_group_id = g.id
WHERE u.uid = "admin";

----- 某个用户组所包含的用户
SELECT g.name, u.uid, u.name
FROM user AS u
JOIN map_user_group ON u.id = map_user_group.user_id
JOIN user_group AS g ON map_user_group.user_group_id = g.id
WHERE u.name = "admin";

以及用户的权限配置。

----- 用户绑定的角色
SELECT u.uid, u.name, r.name
FROM user AS u
JOIN map_user_group ON u.id = map_user_group.user_id
JOIN map_user_role ON map_user_group.user_group_id = map_user_role.user_group_id
JOIN user_role AS r ON map_user_role.role_id = r.id
WHERE u.uid = "admin";

----- 用户绑定的策略
SELECT u.uid, u.name, p.name, p.action, p.resource
FROM user AS u
JOIN map_user_group ON u.id = map_user_group.user_id
JOIN map_user_policy ON map_user_group.user_group_id = map_user_policy.user_group_id
JOIN user_policy AS p ON map_user_policy.policy_id = p.id
WHERE u.uid = "admin";

用户组管理 #

----- 查询某个用户所属的用户组
SELECT u.uid, u.name, g.id, g.name
FROM user AS u
JOIN map_user_group ON u.id = map_user_group.user_id
JOIN user_group AS g ON map_user_group.user_group_id = g.id
WHERE u.uid = "r00258912";

----- 查询某个用户管理的主机组信息
SELECT user.*, host_group.*
  FROM user
  JOIN map_user_group ON user.id = map_user_group.user_id
  JOIN map_user_host_group ON map_user_group.user_group_id = map_user_host_group.user_group_id
  JOIN host_group ON map_user_host_group.host_group_id = host_group.id
  WHERE user.id = 1;
  
----- 查询某个用户管理的主机组信息，同时包含用户组信息
SELECT host_group.*, user_group.*
  FROM user
  JOIN map_user_group ON user.id = map_user_group.user_id
  JOIN map_user_host_group ON map_user_group.user_group_id = map_user_host_group.user_group_id
  JOIN host_group ON map_user_host_group.host_group_id = host_group.id
  JOIN user_group ON map_user_group.user_group_id = user_group.id
  WHERE user.id = 1;

----- 通过用户和主机查询主机信息
SELECT user.*, host.*
  FROM user
  JOIN map_user_group ON user.id = map_user_group.user_id
  JOIN map_user_host_group ON map_user_group.user_group_id = map_user_host_group.user_group_id
  JOIN map_host_group ON map_user_host_group.host_group_id = map_host_group.host_group_id
  JOIN host ON map_host_group.host_id = host.id AND host.name = 'host00'
  WHERE user.id = 1;

----- 通过用户、主机、登录用户名 --> 查询主机、登录信息
SELECT user.*, host.*, host_login_user.*
  FROM user
  JOIN map_user_group ON user.id = map_user_group.user_id
  JOIN map_user_host_group ON map_user_group.user_group_id = map_user_host_group.user_group_id
  JOIN map_host_group ON map_user_host_group.host_group_id = map_host_group.host_group_id
  JOIN host ON map_host_group.host_id = host.id AND host.name = 'host00'
  JOIN map_login_group_user ON map_user_host_group.host_group_id = map_login_group_user.host_group_id
  JOIN host_login_user ON map_login_group_user.user_id = host_login_user.id AND host_login_user.login = 'root'
  WHERE user.id = 1;

----- 通过用户和主机查询主机、登录信息，因为主机名基本唯一，可以唯一确定
SELECT user.*, host.*,
  FROM user
  JOIN map_user_group ON user.id = map_user_group.user_id
  JOIN map_user_host_group ON map_user_group.user_group_id = map_user_host_group.user_group_id
  JOIN map_host_group ON map_user_host_group.host_group_id = map_host_group.host_group_id
  JOIN host ON map_host_group.host_id = host.id AND host.name = 'host00'
  JOIN map_login_group_user ON map_user_host_group.host_group_id = map_login_group_user.host_group_id
  JOIN host_login_user ON map_login_group_user.user_id = host_login_user.id
  WHERE user.id = 1;

----- 通过用户查询角色和策略信息，角色和策略与用户组绑定
SELECT user.*,
  FROM user
  JOIN map_user_group ON user.id = map_user_group.user_id
  JOIN map_user_role ON map_user_group.group_id = map_user_role.user_group_id
  JOIN role ON map_user_group.role_id = role.id
  WHERE user.id = 2;
